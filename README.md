# NebulaForge QoL Modpack

NebulaForge is a quality-of-life modpack for the fictional game **Astral Foundry**. It adds inventory sorting, shader presets, crash-safe autosaves, and multiplayer lobby fixes.

## Quick install

1. Install Astral Foundry `v4.8.x`.
2. Copy this folder into the game's `mods/` directory.
3. Run the launcher once.
4. If the launcher fails, follow `INSTALL.md` and open a compatibility report using the provided issue template.

## Known issue

Some launcher environments fail to load `nebula_renderer_patch.afmod` when shader cache state is inconsistent. Maintainers need a complete compatibility report to reproduce the problem.

## Support

Reports should include launcher version, modpack version, relevant crash output, compatibility matrix, and the maintainer provenance section described in `INSTALL.md`.
